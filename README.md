# Raksha-Kavach – Worker Safety Auditor App 🦺

Project Overview

Raksha-Kavach is an Android healthcare and worker-safety application developed as part of the MindMatrix VTU Internship Program. The app is designed to improve safety awareness among construction and factory workers, especially in small towns and unorganized sectors where safety measures are often ignored.

The main goal of this project is to educate workers about the importance of Personal Protective Equipment (PPE) and help reduce workplace accidents through visual guidance and safety tracking.

Problem Statement

Many workers neglect wearing safety equipment because they feel it is uncomfortable or believe accidents will not happen to them. In most workplaces, there is no proper visual system to explain the risk level involved in different tasks.

Raksha-Kavach addresses this issue by providing a simple and interactive safety-checking system that visually guides workers about the required protective gear for their job.

Project Vision

Raksha-Kavach acts as a **Worker Safety Auditor**.
The application allows workers or supervisors to select a work task such as welding, trench digging, or height work. Based on the selected task, the app displays the mandatory safety gear needed for that activity.

The app also includes:

* Safety checklist system
* Risk indication feature
* Daily safety quiz
* Near-miss incident logging
* Worker safety score tracking

This helps create awareness and encourages a safety-first mindset.

Features

* ✅ Task-based safety checklist
* ✅ PPE guidance (Helmet, Gloves, Boots, Goggles, etc.)
* ✅ Risk meter for unsafe actions
* ✅ Daily safety reminder and quiz
* ✅ Near-miss reporting system
* ✅ User-friendly Android UI using Jetpack Compose
* ✅ Local data storage using Room Database

App Workflow

1. User selects the work task.
2. App displays required safety equipment.
3. Worker checks completed safety items.
4. Risk meter shows possible injury risk if safety gear is ignored.
5. Safety score updates daily.
6. Near-miss incidents can be recorded for future prevention.
